Maisonuts.

Client background.

Maisonuts is a pastry and cafe business operating online due to MCO. The logo’s nature is themed around Arabic and Malay fusion. The client also stated to include the “batik” design that was handed down from their late mother. This particular pattern is used throughtout the design as way to symbolize Malay culture and a way to express an iconic identity for the company.

This project has improved my understanding of using fonts, creating patterns and using mockups as a way to show the clients how the product would look like in the real world.

One of the main challenges with tweaking the logo was to not overdo and change too much of the old logo. The old logo had already its own style and changing it might cause the logo to lose its identity. Some few minor hanges were done to the “Maisonuts” text. Instead of just using the font and leaving it as is, I chose to add longer strokes to some letters. Using a combination of both lower case and upper case letters to give it a variety of size.

Batik pattern.

The batik pattern was by far one of the hardest thing to vectorie in any of my project. This was such a challenge as it requires me to vectorise this batik pattern in Adobe Illustrator. I’ve searched far and wide for this particullar pattern and how to vectorise Batik pattern. It has come to my attention, that there is no vectorise batik pattern being done on the internet. This became such a hassle as batik is one of those patterns that is best done by hand and hard to imitate. Whether it was possible or not, I had to atleast try and vectorise it. Some elements for the batik can be done using basic shapes such as circles. The problem came when trying to do the brown lines. This requires me to create one singluar stroke using the pen tool and then repeating it 7 times inwards to create this repeatative pattern.
